ENTREGA TEST E2E Y APIS DEVSU
    1. Ejecutar el comando "npm install" que insalara todas las dependencias para ejecutar estas pruebas
    2. Colocar en la consola el comnado "run test"
    3. Despues de la ejecución abrir el reporte que esta en la carpeta "reports", dando clic derecho al archivo index.html y seleccionado la opción show in browser
        - En el reporte pueden ver la ejecución de las dos pruebas cada una en diferentes secciones
        - Para ver la ejecución de una determinada prueba de clic en cada uno de los itos de prueba, por ejemplo "Flujo completo de compra"
        - Al finalizar el reporte de cada ito podra ver el video de cada ejecución, en el caso del flujo completo de compra tambien podra ver capturas de pantalla de las partes mas importantes del proceso
        - En la parte derecha de cada ito podra encontrar el tiempo que tomo la ejecución de la prueba
